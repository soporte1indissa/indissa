Gear System

Es una aplicación que busca satisfacer todas las necesidades administrativas empresariales. Fue desarrollada durante el año 1994 dentro de las instalaciones de INGUASOFT (Industria Guatemalteca de Software) y se basó en el sistema operativo Unix. Durante el año 1997, debido a la implementación de nuevas tecnologías, el sistema migra a un ambiente gráfico desarrollado en Visual Basic, generando así, la nueva versión de un sistema administrativo orientado a cualquier tipo de empresa, convirtiéndose en una herramienta rápida y exacta con la información contable y financiera. Finalmente, se integra el producto Gear online, a partir del año 2007, el cual se desarrolló para apoyar la necesidad de conectividad, así como también, para obtener bases de datos integradas y en línea desde cualquier parte del mundo, convirtiendo a este, en un producto de mayor énfasis en la demanda de software.
